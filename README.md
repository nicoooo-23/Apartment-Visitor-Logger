## Project Structure Overview

The project is organized into iterative drafts to track development progress:

* **AVL:** The current working directory containing the finalized logic and assets.
* **FullStackDraftV1 & V2:** Draft folders where backend logic and frontend processing were developed and refined.
* **assets:** CSS file stored here.
* **includes:** Contains reusable PHP components like headers and navigation bars.
